输入输出样例
样例1

输入: 26
输出: 1a

样例2

输入: -1
输出: ffffffff

解题思路
将这个数对16求余，之后除以16，依次从低位转换。

note:负数的补码处理可以采用 unsigned int x = num 和 x /= 16处理。 也可以采用 num = ~(-num)+1 加上 x >> 4 处理，int 是32位的，每次移动 4 位，一共可以移动 8 次。
